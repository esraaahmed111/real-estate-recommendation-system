# Real Estate Recommendation & Price Prediction System
A **machine learning-powered real estate system** combining a **KNN-based recommendation engine** with **ensemble regression models** for property price prediction on Egyptian real estate listings.

# Machine Learning Models:
  - **KNN**: Property recommendations ranked by cosine similarity score
  - **HistGradientBoostingRegressor**: Price prediction with early stopping
  - **BaggingRegressor**: Ensemble of Decision Trees with OOB scoring
  - **DecisionTreeRegressor**: Base estimator within the Bagging ensemble
